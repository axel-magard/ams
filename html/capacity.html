AMS Capacity Calculation

Capacity Calculation in AMS computes the number of Workcenter needed to fulfill a weekly production demand for all Product Types.

The report will provide

  • Weekly Input for each Product Type
  • Days/Week and Hours/Day as specified as model parameter
  • Daily Production Demand and Input per Product Type
  • Computed and ounded capacity and delta to current model for every Workcenter
