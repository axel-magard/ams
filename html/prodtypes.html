AMS Product Types

A Product Type in AMS represents a type of part manufactured. For all Product Types a Product Mix is defined by specifying a percentage value for each Product Type indicating the share this Product Type has in the overall Workload for the manufacturing line modeled.

For every AMS Product Type Operations and a Process Flow have to be specified.

An AMS Product Type has the following attributes:

  • Parts In: how parts are called when entering the manufacturing line ( e.g. "Wafer" )
  • Parts Out: how parts are called when leaving the manufacturing line ( e.g. "Slider" )
  • Description: description for this Product Type
  • Percentage share of this Product Type in the overall Workload for the manufacturing line modeled.

⚠️Please note that all Percentage values for all Product Types need to add up to 100% for a valid model. If this is not the case an appropriate error message will be shown when trying to compute this model.

In addition the following cost and resource specific attributes can be defined:

  • Inventory: Inventory cost per part ( capital commitment )
  • Space needed to store a part of this Product Type
  • Income Income generated when selling this part

To change attributes of a Product Type either right-click on a Product Type in the tree view of model artefacts on the left side of the screen and select "Edit" ...

The "Edit Product Types" dialog also allows to add or delete Product Types. This can also be achieved by right-clicking on a Product Type and selecting "Add" or "Delete".

When deleting a Product Type a confirmation message box will appear reminding you that also all operations defined for that Product Type will be deleted.
