Welcome to AMS - The Analytical Model System !

AMS - The Analytical Model System - can be used to plan manufacturing lines and determine key performance indicators like

  • Maximum Throughput
  • Utilization
  • Leadtime
  • Work In Process (WIP)
  • Efficieny
  • Yield
based on a model consisting of
    Workcenter
    physical resources like e.g. machines, where work is performed
    Product Types
    parts manufactured
    Operations
    process steps needed to manufacture parts
    and a Process Flow
    describing how parts flow from operation to operation

For a given model AMS allows to compute

Start Factors
for a selected Product Type and a Daily Output to be specified
Capacity Calculation
for a given Weekly Demand to be specified for all Product Types. The Capacity Calculation basically will show how many of each Workcenter are needed to fullfil the given Weekly Demands.
Line and Workcenter Characteristics
for all or a specifc Product Type, computing results like Utilization, WIP or Leadtime for the overall manufacturing line as well as each Workcenter
Line and Workcenter Plot
for all or a specifc Product Type, computing results like Utilization, WIP or Leadtime for the overall manufacturing line as well as each Workcenter, ploted over Arrival or Departure Rate of parts, or number of parts started or shipped.
