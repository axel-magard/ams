AMS Characteristics Calculation Table

Characteristics Calculation in AMS computes essential key performance indicators for the entire manufacturing line as well as for every single Workcenter

When invoking this report a dialog will ask for a Product Type to be selected, or "Overall" to compute for all Product Types, and an Arrival Rate, Departure Rate, Daily Starts (SOB) or Daily Shipments (SHIP), which needs to be below the maximum possible production volume computed by AMS and shown in this dialog.

The report will provide for the entire manufacturing line:

Maximum Arrival Rate
Maximum number of parts per unit of time possible for this manufacturing line to be started
Parts Started
Number of parts per unit of time started for this manufacturing
Good Parts Out
Number of parts per unit of time shipped from this manufacturing line
Yield
Percentage of good parts shipped related to parts started
Days / Week
Days per Week assumed for this computation
Hours / Day
Hours per Day assumed for this computation
Maximum thoughput (shipment)
Maximum number of parts possible to be shipped from this manufacturing line
Daily going rate (start of build):
Number of parts started per day
Daily going rate (shipment)
Number of parts shipped per day
Takt time
Time interval between parts travelling through manufacturing line
Utilization (average)
Average utilization of all workcenter
Raw cycle time (all parts)
Minimum possible processing time (without any waits)
Leadtime (all parts)
Actual processing plus waiting time
Leadtime (including idle periods):
Actual processing plus waiting time plus idle time of manufacturing, based on Days / Week and Hours / Day specified
Line Efficiency
Percentage of Raw Cycle Time related to Leadtime
Line Efficiency (including idle periods)
Percentage of Raw Cycle Time related to Leadtime including idle periods
WIP
Work In Process: average number of parts present in manufacturing time - in process or waiting

The report will provide for every workcenter:

Overall load
Parts processed at that workcenter per unit of time
Net. Utilization
Utilization at that workcenter
Tot. Utilization
Utilization at that workcenter including down times
Leadtime
Actual processing plus waiting time at that workcenter
WIP
Work In Process: average number of parts present at this workcenter - in process or waiting
Queue Length Mean
Average number of parts waiting in front of at this workcenter
Queue Length 90 % Quantil
Threshold of number of parts waiting in front of at this workcenter not exceeded with a probability 90 %. This value is useful to size buffers needed in front of that workcenter.

In addition the following cost values can be computed for every workcenter:

Depreciation
Share of the yearly Depreciation cost of that workcenter occuring per part
Service & Maintenance
Share of the yearly Service & Maintenance cost of that workcenter occuring per part
Operation Expense
Cost for processing one part at given workcenter
